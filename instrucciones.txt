1. 	Clonar el repositorio
    git clone <URL-del-repo>
    cd RateLimingProject

2. 	Instalar dependencias
    npm install

3. 	Crear/modificar el archivo  en la raíz del proyecto
    DATABASE_URL="postgresql://postgres:12345@localhost:5432/ratelimit_db?schema=public"
    SESSION_SECRET="clave_segura_para_sesiones"
    PORT=3000

4. 	Configurar PostgreSQL localmente
• 	Instalar PostgreSQL (si no lo tienen).
• 	Crear la base de datos(o abrir pgadmin):
    CREATE DATABASE ratelimit_db;

5. Ejecutar migraciones de Prisma
    npx prisma migrate dev --name init

6. Generar el cliente de Prisma
    npx prisma generate

7. Levantar el servidor de desarrollo
    npm run dev
